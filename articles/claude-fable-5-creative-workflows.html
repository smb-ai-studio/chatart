EVALUATION CRITERIA

What creative teams should test first.

Brief comprehension

Can the model turn a messy launch brief into audience, positioning, channel plan, creative route, and review criteria without losing constraints?

Brand voice consistency

Can it keep one tone across scripts, captions, ad copy, landing-page headlines, and creator talking points?

Editorial judgment

Does it flag unsupported claims, risky wording, weak proof, unclear CTAs, and platform mismatch before assets go live?

Production handoff

Can it convert strategy into prompts that video, image, copy, and music tools can actually use in ChatArt Pro?

WORKFLOW

A safe Claude Fable 5 evaluation flow.

  1. Create a fixed test brief. Use one product, one audience, one offer, one brand tone, and one set of channels so outputs can be compared fairly.
  2. Ask for a campaign system. Request positioning, asset list, prompts, captions, ad angles, landing-page message, and QA checks in one response.
  3. Score the output. Grade clarity, consistency, usefulness, factual caution, and how easy the plan is to execute in ChatArt Pro.
  4. Run a second-pass editor prompt. Ask the model to critique its own plan and identify risks before production.
  5. Compare against your current model. Use the same brief and scoring rubric. A new model is only better if it improves the actual workflow.
  6. Document what changed. Keep dated notes on model behavior, access, limits, and source URLs. This is important for fast-moving releases.

EXAMPLE TEST BRIEF

Use one controlled brief to avoid subjective comparisons.

Product: a mobile app that helps independent fitness coaches turn client notes into weekly workout plans. Audience: solo coaches with too many manual follow-ups. Promise: spend less time formatting plans and more time coaching. Channels: Reels, TikTok, paid social, landing page, and email. Tone: practical, energetic, trustworthy.

Ask Claude Fable 5 and your current model to create the same campaign asset system from this brief. Then compare the outputs using the matrix above. The best model is the one that gives you clearer prompts, fewer unsupported claims, and a stronger production handoff.

PROMPTS TO COPY

Evaluate the model with practical production prompts.

Campaign system prompt

Act as a campaign strategist and production editor. Turn this brief into a connected asset system: hero video, product visual, thumbnail, captions, paid-social hooks, landing-page angle, music direction, and QA checklist. Keep every asset tied to the same promise.

Brand voice stress test

Rewrite this campaign message for three channels: TikTok creator post, paid social ad, and landing-page hero. Preserve the factual claims and offer. Do not add proof that is not in the brief.

Risk review prompt

Review this campaign pack for unsupported claims, vague CTAs, channel mismatch, visual inconsistency, compliance risk, and confusing audience assumptions. Return a table of issues and fixes.
